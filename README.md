# Tic-Tac-Toe
Tic-Tac-Toe game built with React/Redux

## Installing
* Clone the repo
```
git clone https://github.com/ssaitta/Tic-Tac-Toe.git
```
* Install dependencies
```
npm install
```
* Start server
```
npm run start-dev
```
* Navigate to webpage at http://localhost:8080/

## Built With
* React
* Redux
* Node.js
